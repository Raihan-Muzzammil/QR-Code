Hello

Today we are gonna be learning how to make a (somewhat good) webpage with your notepad and keyboard.

Step: 1 (Declaring your Document)

Open Notepad and begin by typing "<!DOCTYPE html>".

Typing this declares your document as a HTML Document.

Step: 2 (Starting up)

There are certain Keywords or Tags which are important for just basic display

A HTML document always begins with "<html>" and ends with "</html>". Note that I use a forward slash which denotes that its the end. You will use it more often when making a webpage.

Step: 3 (Giving the Webpage a Body)

This is the part where the magic happens.

After typing "<html>" type "<body>". When you do this you essentially start making your main display for your webpage.
Just like "<html>" and "</html>" you start your webpage's content with "<body>" and end with "</body>".

Note: Anything between the tags "<body>" and "</body>" will appear on your browser.

Step: 4 (Giving the body some soul)

Now that you have started with the body, time to add a little info and everything else that you want to.

This part is what will appear on your Web Browser , So the main focus will be on this part.

Let's look at the basic tags.

The Heading :

To add a heading type "<h1>" and end with "</h1>." The "1" in "h1" denotes the main heading and also determines the size of the heading.

The Higher the Number the smaller the size.

The Paragraph :

When you have a heading , there should be some information below it. So we categorize them into paragraphs.

To begin a Paragraph we type "<p>" and end the paragraph with "</p>.".

Remember that whenever you want to close a tag you have to add "/" and type the tag name you began with.
Example : "<p>" and "</p>".

You can use"<br>" to break and start from a new line in the same paragraph
You can also use "<b>" at the start and "</b>" at the end to bold the letters or a sentence.

The Decorations :

Now that we know to make a webpage , you can go ahead and write stories if you want to.
But its still missing the decorations and fancy stuff to make it look beautiful.

Therefore lets start by adding decorations like :

Links :

Links can allow you to share some other webpages that you might like or some image that you were amazed by.

And for no reason , here is a picture of a cat.

To hyperlink a link to a word or sentence simply type the tag "<a href=>" followed by your word or sentence and then end with "</a>."

Images :

A Web Page becomes colourful when you add an image or two to it or atleast thats what I believe.
So here is another image of a cat

That's Better.

To add an image type the tag "<img src="(file name)" alt="(alternative file name)" width="(width)" height="(height)">"

Buttons :

People like clicking on things that look like they want to be clicked.
Buttons look like they want to be clicked and hence invite people to click.

Therefore lets make a button which does absolutely nothing.

To make a button type the Tag "<button>" followed by the text to be represented on the button and end it with "</button>".



These are some of the many ways you can decorate your webpage.

Step: 5 (Packing up)

After you are done with the body of your webpage be sure to use the tag "</body>" and "</html>" to complete your webpage.

Then use Ctrl+Shift+S to save your file as .htm or html file (it doesn't matter which one) and the encoding as UTF-8 since that is widely used for HTML Documents
